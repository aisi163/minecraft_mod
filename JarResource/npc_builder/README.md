<h2>版本使用说明</h2>
<ul style="line-height: 1.8; margin: 10px 0 20px 20px;">
  <li>仅推荐使用<strong>未弃用版本</strong>，弃用版本存在已知BUG/兼容性问题，不保证正常运行；</li>
  <li>不同Java版本需匹配对应Minecraft版本，避免因环境不兼容导致模组启动失败；</li>
  <li>双端模组需同时部署至客户端和服务端中，单端模组仅需部署至对应端（客户端/服务端）；</li>
  <li>如需二次开发，可基于v1.0.0版本源码修改，遵循MIT开源协议保留原作者声明。</li>
</ul>
